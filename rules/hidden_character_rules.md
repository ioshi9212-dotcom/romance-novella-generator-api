# Hidden Character Rules

Если персонаж важен, но POV его ещё не знает, заранее сохраняется только seed.

Полная анкета создаётся при первом сценическом появлении или при реальном источнике информации.

Seed не должен содержать:
- полную внешность;
- имя, если оно должно быть скрыто;
- тайну;
- готовую романтическую динамику.
