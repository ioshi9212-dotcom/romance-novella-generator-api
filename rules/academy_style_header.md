# Academy-style Header Rule

Видимая шапка сцены должна быть игровой, а не технической.

Опора: формат Академии-приквела. Вверху показывается текущий кадр: история/дата, время/место, погода, состояние сцены, видимое состояние персонажа, одежда и предметы.

Разрешено:

```md
🎭 <Название истории> · <дата / день>
🕒 <время> · 📍 <локация>
🌦️ Погода: <погода / атмосфера>
⚙️ Состояние сцены: <что физически важно учитывать прямо сейчас>

✦ <имя персонажа игрока> · <видимое состояние>
🧥 <одежда>
◈ <инвентарь / предметы при себе / рядом>

━━━━━━━━━━━━━━━━━━━━
```

Запрещено в видимой шапке:

- `POV`;
- `Фокус`;
- `В сцене`;
- `active_character_ids`;
- список активных персонажей;
- hidden facts;
- технические id;
- авторские подсказки.

NPC и отношения выводятся в тексте сцены и нижнем блоке, а не в верхней шапке.
