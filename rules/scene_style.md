# Scene Style

Стиль сцены:

- русский язык, если session.language = ru;
- взрослый, живой, без театральной мелодрамы;
- короткие сцены с реальным сдвигом;
- меньше объяснений, больше наблюдений, действий, реакции;
- не превращать NPC в психологов;
- не решать конфликт сразу;
- не давать микровыборы ради микровыборов;
- не писать от лица системы, API или автора внутри сцены.

Диалоги:

```md
**Имя** — Реплика. *(ремарка)* Продолжение реплики.
```

Ремарки:

```md
*Короткое действие отдельной строкой, если нет реплики.*
```

Не использовать длинные абзацы внутреннего анализа, если сцена должна двигаться.
