# Knowledge Rules

Каждый персонаж знает только то, что:

- видел;
- слышал;
- ему сказали;
- он мог честно вывести из видимых фактов;
- уже записано в его knowledge.

Глобальный state не равен знанию NPC.

Если персонаж отсутствовал в сцене, он не узнаёт факт без:
- звонка;
- сообщения;
- свидетеля;
- отчёта;
- публичного события.
