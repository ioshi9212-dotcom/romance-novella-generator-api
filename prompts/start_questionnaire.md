# Start Questionnaire Template

Когда пользователь пишет только «начнём», «начнем», «старт», «создай сессию» или «новая игра» без каких-либо пожеланий, верни короткую анкету вместо создания случайной истории.

Анкета должна быть простой. Пользователь может поставить прочерк, если хочет, чтобы ИИ придумал сам.

## Видимый текст анкеты

```md
Заполни коротко. Где не важно — ставь прочерк, я придумаю.

1. Жанр / поджанр:  
   например: романс, академия, мистика, детектив, фэнтези, постапок, офис, городское фэнтези.

2. Персонаж игрока / за кого играем:  
   женщина / мужчина / другое / придумать.

3. Возраст и общий тип героя:  
   например: 25 лет, закрытая, усталая, язвительная, мягкая, опасная, обычная.

4. Сеттинг:  
   где начинается история: город, академия, база, деревня, остров, офис, чужой дом, дорога.

5. Главная завязка:  
   что примерно должно случиться в начале.

6. Романтика / отношения:  
   бывшие, враги, друзья детства, начальник/подчинённая, вынужденный союз, медленный burn, без романтики.

7. Тон:  
   мрачный / спокойный / напряжённый / взрослый / с юмором / без сахара.

8. Что нельзя:  
   темы, жанры, типажи, которые не нужны.

9. Нужная арка, если есть:  
   например: возвращение домой, выживание, расследование, обучение, месть, исцеление, побег.

10. Имена / стиль имён, если важно:  
   по умолчанию не русские: западные, японские, англо-японские, фамилии тоже не русские. Можно прочерк.

11. Особые механики истории:  
   например: магия, репутация, болезнь, долг, тайна, опасность, рейтинг отношений. Можно прочерк.
```

## Правило

Если пользователь дал хотя бы одно содержательное пожелание или уже ответил на анкету — не задавай её повторно, сразу создавай bootstrap.
Ответ может быть частичным. Пропущенные пункты, прочерки и просьба «придумай сам» означают, что ИИ должен самостоятельно заполнить недостающее подходящими деталями.


## После анкеты

Когда пользователь ответит одним сообщением, передай весь его текст дословно в raw_start_text и создай preview: анкета главной героини, известные персонажи, стартовая ситуация и story_plan. Сцену писать только после подтверждения пользователя.
