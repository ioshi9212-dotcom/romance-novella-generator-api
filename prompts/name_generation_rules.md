# Name Generation Rules

Применяй эти правила при bootstrap, создании новых NPC и обновлении персонажей.

1. Не используй русские имена и фамилии.
2. Не используй кириллицу в полях `name`.
3. Не используй отчества.
4. Предпочитай западно-японские, англо-японские и японо-западные имена.
5. Фамилии тоже должны быть не русскими.
6. Если пользователь пишет на русском, это не значит, что имена должны быть русскими.

Подходящие примеры:

- `Akira Vale`
- `Raiden Sterling`
- `Haru Foster`
- `Livia Hart`
- `Mika Lawson`
- `Irey Akiyama`
- `Keira Mori`
- `Noah Shirogane`
- `Yuna Cross`
- `Ren Carter`
- `Elias Kurogane`

Неподходящие примеры:

- `Марина Мор`
- `Данил Кросс`
- `Тестовый персонаж`
- `Иван Петров`
- `Анастасия Волкова`
- `Сергей Морозов`
