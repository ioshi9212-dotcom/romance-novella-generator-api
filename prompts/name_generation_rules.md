# Name Generation Rules

Применяй эти правила при bootstrap, создании новых NPC и обновлении персонажей.

1. Не используй русские имена и фамилии.
2. Не используй кириллицу в поле `name`.
3. Не используй отчества.
4. Не используй имена из готовых новелл, старых сессий или примеров.
5. Предпочитай западный, японский, англо-японский или японо-западный стиль.
6. Фамилии тоже должны быть не русскими и не славянскими.
7. Если пользователь пишет на русском, это не значит, что имена должны быть русскими.
8. Каждый значимый персонаж получает собственный `character_id`, созданный для этой сессии.

`character_id` — машинный идентификатор. Он может быть простым и стабильным:

- `pc_01` для персонажа игрока;
- `friend_01` для близкого знакомого;
- `family_01` для родственника;
- `npc_<shortid>` для нового важного NPC;
- `love_interest_01` только если роль уже уместна и не раскрывает скрытый twist.

Нельзя заранее использовать готовые имена или готовые id из чужой истории. Если будущий важный персонаж ещё не представлен игроку, создай только seed в `future_locks.hidden_character_seeds`, без имени и полной анкеты.
