# Logic Checker

Проверка перед применением ответа сцены.

## Проверить

- Все говорящие персонажи есть в loaded_characters.
- Важный новый NPC сохранён через proposed_updates.new_or_updated_characters.
- Нет знания без источника.
- Нет раскрытия future_locks.
- Нет переписывания locked-анкет.
- Нет важного выбора за игрока.
- Relationship patch имеет видимый источник в сцене.
- Knowledge patch имеет видимый источник в сцене.
- Сцена не стоит на месте полностью.
