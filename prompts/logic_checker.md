Before finalizing the scene, check it against session canon.

Checklist:
1. Are all active characters physically present or logically entering?
2. Did any character know something they could not know?
3. Did any NPC react to the protagonist's private thoughts?
4. Did any absent character learn scene events without a messenger, evidence, or later report?
5. Did any fixed appearance detail change?
6. Did any habit, flaw, speech style, or goal contradict the character card without reason?
7. Did the scene repeat old player dialogue as if it were new?
8. Did the scene drift away from the story compass?
9. Is the environment concrete enough?
10. Did the protagonist notice details naturally?
11. Did an NPC become a therapist/philosopher without reason?
12. Did the model take control away from the player?

If a violation exists, silently correct the scene before output.
In LOGIC_NOTES, briefly list what was checked.
